QLite

Promises

QLite is a tiny JavaScript Promise library. Promises allow you to deal with asynchronous APIs in a mantainable way.

Download View on GitHub

Example


asyncTask1(asyncTask1_input).
then(function (asyncTask1_output) {
	return asyncTask2(asyncTask1_output);
}).
then(function (asyncTask2_output) {
	// Do something with asyncTask2_output
}).
fail(function (reason) {
	// Single failure handler for any of the previous tasks
});
